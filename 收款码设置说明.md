# 如何设置你的收款码

## 📱 步骤1：生成收款码

### 支付宝收款码
1. 打开支付宝APP
2. 点击首页的"收钱"
3. 保存收款码图片到手机

### 微信收款码
1. 打开微信APP
2. 点击右上角"+" → "收付款" → "我要收款"
3. 保存收款码图片

### 云闪付收款码
1. 打开云闪付APP
2. 点击"收款"
3. 保存收款码图片

## 🌐 步骤2：上传收款码图片

有两种方式：

### 方式A：放在项目的 public 文件夹
1. 将收款码图片保存为 `alipay-qrcode.jpg`, `wechat-qrcode.jpg`, `unionpay-qrcode.jpg`
2. 复制到 `public` 文件夹
3. 在 `recharge.js` 中修改：
```javascript
const PAYMENT_QRCODES = {
    alipay: '/alipay-qrcode.jpg',
    wechat: '/wechat-qrcode.jpg',
    unionpay: '/unionpay-qrcode.jpg'
};
```

### 方式B：使用图床（推荐）
1. 上传到免费图床（推荐：https://imgbb.com/ 或 https://sm.ms/）
2. 获取图片链接
3. 在 `recharge.js` 中修改：
```javascript
const PAYMENT_QRCODES = {
    alipay: 'https://i.ibb.co/xxxxx/alipay.jpg',      // 你的支付宝收款码链接
    wechat: 'https://i.ibb.co/xxxxx/wechat.jpg',      // 你的微信收款码链接
    unionpay: 'https://i.ibb.co/xxxxx/unionpay.jpg'   // 你的云闪付收款码链接
};
```

## ✅ 步骤3：测试

1. 刷新充值页面
2. 选择充值金额（如¥5）
3. 选择支付方式
4. 点击"生成收款码"
5. 看到你的收款码显示
6. 用手机扫码支付
7. 支付完成后点击"已完成支付"
8. 余额自动到账

## 💡 注意事项

1. 收款码图片建议大小：300x300像素
2. 图片格式：JPG 或 PNG
3. 确保图片URL可以公开访问
4. 建议使用稳定的图床服务

## 🔒 安全提示

- 收款码是公开的，任何人都可以扫码给你转账
- 建议定期更换收款码
- 不要在收款码上附带敏感信息
